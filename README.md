# Terminal OS
A small OS I made. Yeah that's it.

# How to Compile
```
make
make run
```
You'll need QEMU and NASM.

If it doesn't work, try compiling it again.

# Supported Commands
`clear` - Clears the terminal

`exit` - Exits the shell

`echo <argument>` - Prints `<argument>` to the terminal

`read <prompt>` - Gets user input
